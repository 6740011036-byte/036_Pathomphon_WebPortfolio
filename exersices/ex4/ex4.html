Ex4 : HTML : Beginner : สร้างหน้าเว็บส่วนตัวอย่างง่ายด้วย HTML

โจทย์นี้จะให้คุณสร้างหน้าเว็บ HTML ง่ายๆ เพื่อแสดงข้อมูลส่วนตัวเบื้องต้นของคุณ เช่น ชื่อ รูปภาพ ข้อมูลแนะนำตัว และข้อมูลติดต่อพื้นฐาน
ถือเป็นการฝึกใช้แท็ก HTML พื้นฐานที่จำเป็น

สิ่งที่โจทย์ต้องการ :
• ใช้แท็ก <h1> สำหรับแสดงชื่อเต็มของคุณ
• ใช้แท็ก <p> อย่างน้อยหนึ่งย่อหน้าเพื่อแนะนำตัวสั้นๆ
• แสดงรูปภาพของคุณ (หรือรูปภาพ Placeholder) โดยใช้แท็ก <img>
• สร้างรายการข้อมูลติดต่อ (เช่น อีเมล, เบอร์โทรศัพท์) โดยใช้แท็ก <ul> และ <li>
• เพิ่มลิงก์ไปยังโปรไฟล์โซเชียลมีเดียที่คุณชื่นชอบ (เช่น Facebook, LinkedIn) โดยใช้แท็ก <a>
