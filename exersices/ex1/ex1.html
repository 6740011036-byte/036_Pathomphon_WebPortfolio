Ex1 : HTML : Beginner : สร้างการ์ดข้อมูลส่วนตัวด้วย HTML

โจทย์นี้จะให้คุณสร้างหน้าเว็บง่ายๆ ที่แสดงข้อมูลส่วนตัวของคุณในรูปแบบของการ์ด โดยใช้โครงสร้าง HTML พื้นฐาน
ฝึกการใช้แท็ก HTML ที่สำคัญเพื่อจัดเรียงและแสดงเนื้อหาต่างๆ

สิ่งที่โจทย์ต้องการ :
• สร้างหัวข้อหลัก (h1) สำหรับชื่อของคุณ
• แสดงรูปภาพของคุณ (หรือรูปภาพใดๆ ที่คุณเลือก) โดยใช้แท็ก img
• แสดงข้อมูลติดต่อ เช่น อีเมลและเบอร์โทรศัพท์ โดยใช้แท็ก paragraph หรือ list
• แสดงคำอธิบายสั้นๆ เกี่ยวกับตัวคุณ (เช่น งานอดิเรก หรือความสนใจ) โดยใช้แท็ก paragraph
• จัดโครงสร้างข้อมูลทั้งหมดให้อยู่ภายใน <div> ที่มี id เป็น "profile-card"
