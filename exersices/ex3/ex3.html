Ex3 : HTML : Beginner : สร้างหน้าโปรไฟล์ส่วนตัวอย่างง่าย HTML

คุณจะต้องสร้างหน้าเว็บ HTML อย่างง่ายเพื่อแสดงข้อมูลโปรไฟล์ส่วนตัวของคุณ ซึ่งประกอบด้วยรูปภาพ ชื่อ อายุ
และรายการความสนใจต่าง ๆ โดยใช้แท็ก HTML พื้นฐาน

สิ่งที่โจทย์ต้องการ :
• ต้องมีหัวข้อหลัก (Heading) สำหรับชื่อโปรไฟล์ของคุณ (เช่น 'โปรไฟล์ของ [ชื่อของคุณ]') โดยใช้แท็ก <h1>
• ต้องมีรูปภาพโปรไฟล์ของคุณ (หรือรูปภาพใดก็ได้) โดยใช้แท็ก <img>
• ต้องมีข้อมูลอายุ โดยใช้แท็ก <p>
• ต้องมีรายการความสนใจอย่างน้อย 3 รายการ โดยใช้แท็ก <ul> และ <li>
• เนื้อหาทั้งหมดควรอยู่ภายในแท็ก <body>
